System Requirements

Minimum

  • Ubuntu 20.04 or Debian 11
  • 4 GB RAM
  • 2 GB free disk space
  • Internet connection (for downloads)

Recommended

  • Ubuntu 24.04 or Debian 12
  • 8 GB RAM or more
  • 5 GB free disk space
  • Sudo/root privileges

Note: LocalForge requires sudo/root privileges to install development tools (PHP, Nginx, databases, etc.).

Quick Installation Guide

1

Download LocalForge

Choose your preferred package format (.deb or AppImage) and download it.

2

Install the Package

For .deb package:

sudo dpkg -i localforge_{{VERSION}}_amd64.deb

For AppImage:

chmod +x LocalForge-{{VERSION}}.AppImage
3

Launch LocalForge

Open LocalForge from your applications menu or run the AppImage file.

4

Install Development Tools

Go to "Manage Services" tab and install PHP, Node.js, Nginx, and other tools you need.

5

Start Creating Projects

You're ready! Start creating Laravel, Vue, React, or WordPress projects with one click.
